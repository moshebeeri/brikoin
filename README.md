Try it at: [https://redux-saga-firebase.firebaseapp.com/](https://redux-saga-firebase.firebaseapp.com/)

## How to:

- install: `yarn`
- run: `yarn dev`
- build: `yarn build`
- deploy: `yarn deploy`
